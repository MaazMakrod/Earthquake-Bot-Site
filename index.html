How it Works

1. Volcano Discovery

Quake Bot retrieves real-time earthquake statistics from VolcanoDiscovery.com.

The site provides data on the most recent earthquakes worldwide. Quake Bot scrapes this data and presents it in an easy-to-digest format for its Discord users.

2. Beautiful Soup 4

To scrape the data, we have used Python's BeautifulSoup4 library. This library allows us to search a site's HTML code for specific tags and get the data that is present in them. After inspecting Volcano Discovery's source code, we were able to narrow down on th elocation of the infromation we need to present and extract it.

3. Discord API

Using Discord's Developer Portal we are able to create a bot that ouputs this information to you in Discord! This is done by importing Discord's discord Python library, allowing us to use functions that create the bot, run the bot, and respond to certain events!
